### description

A simple Portfolio template for developer/designers built with React. 

### [live preview]



### Features

- Fully Responsive
- Multi-Page Layout
- Contact Form With EmailJs
- React-Bootstrap



### Setup

Get the code.

 
 
Install required dependencies

<pre>yarn install</pre>

modify pages content in  `src/content_option.js`.

statrt the server

<pre>yarn start</pre>
